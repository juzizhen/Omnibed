# OmniBed | 拼好床 🛏️

打破原版 Minecraft 只能“规规矩矩”睡觉的限制！**OmniBed** 允许玩家在多张相连的床上以任意方向自由入睡，为你和朋友在联机时带来最真实的“大床翻滚”体验。

## ✨ 核心特性

* **全向自由睡姿**：支持跨越两张相邻的床入睡。无论是横着睡还是斜着点，只要你的视线方向（前、后、左、右）有连续的两格空床位，身体便会自动顺应床铺走向完美躺下。
* **智能床位分配**：联机不再抢床头！如果你点击的床位已经被室友占用，系统会自动环顾四周，把你智能“塞”进旁边相邻的空床位里。
* **双重重生点保护**：你的重生点将同时绑定在你入睡时占用的**两个**床方块上。即使其中半张床被不慎破坏，只要另一半还在，你的重生点就依然有效！
* **完美的多人同步**：基于原生实体数据轨道（DataTracker）同步，其他玩家能够完美看到你“横七竖八”的真实睡姿，绝不穿模。
* **极简纯净**：极致轻量化设计，完美契合原版生存体验。

## 🎮 使用方法

与原版完全一致，没有任何学习成本：
1. 将两张或多张床紧挨着摆放（颜色不限，怎么拼都可以）。
2. 站在床边，朝着你想要躺下的方向**右键点击**。
3. 享受你的大床！

*(注：如果你睡到一半，身下的任意一个床方块被破坏，你会被安全地唤醒。)*

## 📦 环境需求

* **运行环境**: Fabric 
* **游戏版本**: 1.20.1
* **运行依赖**: Fabric api
* **安装要求**: 需要在**客户端**与**服务端**同时安装此模组，以确保动画同步与物理判定生效。
